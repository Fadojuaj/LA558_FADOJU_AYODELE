Creating Web Maps in R

Leaflet Maps with Markers


Plot 1: Vacant Housing Stock in Southwestern Parts of Nigeria

This map displays circle markers for the cities in Southwest Nigeria. It includes popups for the city, housing unit, and monthly rent range.


Plot 2: Choropleth Map Showing Secondary Hospitals in Nigeria

This is an interactive map showing the number of secondary hospitals in each state of Nigeria.

It includes a legend and a popup.The data set was obtained from Nigeria Health Facility Registry (HFR) .

Plot 3: Occupied and Vacant Housing Stock Across Southwestern Parts of Nigeria

This map shows clusters of markers in Southwestern Nigeria. It provides a clean interface for reading the map.
